Space Hub

A simplified solar system simulation

October 28, 2020 Kyra ZHOU

An Intro to Space Hub program:

Computer Science is a facinating subject, and one of the reasons is that it can interact with various subjects to solve problems . In order to help GCSE students learn the gravitational field and the astronomy better, a simulation of the solar system will be handy.

To simulate the solar systems, I need to figure out how to apply the physics equations about the gravitational field onto the rotating planet to keep it on track and updating its parameters (i.e., linear and angular velocities and acceleration). This problem is a standard computational problem since there is no way to simulate a solar system accurately manually, and the simulation requires tons of repeated calculations which the computer is good at. Hence, it is sensible to design a computational solution to solve this demanding computational task

In this project - Space Hub, my objective is to build up a program which contains two parts:

  • The first part will display the rotation of each planet in our solar system and their interesting facts (i.e., Mercury, Venus, Earth, Mars, Jupiter, Saturn, Uranus and Neptune).
  • The second part will be a control part, users can change the velocity of the planet and see what will happen as a result.
The project aims to help GCSE Physics students understand the underlying physics theorem better through interactions with the program.

How can I play the little simulation?

The program is available on Github and you can clone the file and run it on your own computer!
